Bloc Jams

Bloc Jams is a single-page, streaming music application. It was created using React, HTML, CSS and JavaScript. Bloc Jams allows users to pick an artist and listen to songs on their album.

PA Giving Day Landing Page

This page was created as a landing page while waiting for a campaign to launch. Because of time restrictions I used a Bootstrap framework and used JavaScript to create a countdown clock to generate excitement.

Bloc Chat

Bloc Chat was created using React, Firebase, HTML, CSS and JavaScript. Users are able to sign in using their Gmail account, create new chat rooms and send messages in different chat rooms.

Bloccit

Bloccit is a Reddit-like application created using Node.js. This app has many features such as authentication, authorization, creating topics, posting and commenting. Bloccit also incorporated Test Driven Development, PostgreSQL and Express, among others.

Blocipedia

Blocipedia is an application that allows users to create, collaborate and share wikis. It was created using Node.js, PostgreSQL and Express. I integrated Stripe to charge users for premium accounts, and implemented email sending and user authentication using external APIs (SendGrid and Passport.js).
